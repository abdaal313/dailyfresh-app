📋 How to Fix "Tracking Prevention" Error:

🦊 Firefox:
  1. Type about:config in address bar
  2. Search for privacy.trackingprotection.enabled
  3. Set it to false
  4. Refresh the page
🧭 Safari:
  1. Go to Preferences → Privacy
  2. Uncheck "Prevent cross-site tracking"
  3. Refresh the page
🔵 Chrome/Edge:
  1. Go to Settings → Privacy and security
  2. Change "Cookies and other site data" to allow all
  3. Refresh the page
